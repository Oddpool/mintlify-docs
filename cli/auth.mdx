---

## Login

```bash
benchspan login
```

Opens your browser to authenticate. Once logged in, the CLI stores your session locally at `~/.benchspan/config.json`.

For CI/headless environments:

```bash
benchspan login --email you@company.com --password "..."
```

## Logout

```bash
benchspan logout
```

Clears saved credentials.

## Whoami

```bash
benchspan whoami
```

Shows your current login status and API endpoint.
