---
title: "benchspan agents"
description: "List available built-in agents."
---

## Usage

```bash
benchspan agents
```

## Output

Shows a table of built-in agents with their required and optional env vars:

| Name | Description | Env Vars |
|---|---|---|
| `claude-code` | Claude Code | `ANTHROPIC_API_KEY`, `BENCHSPAN_MODEL` (optional), `BENCHSPAN_MAX_TURNS` (optional) |
| `openhands` | OpenHands | `LLM_API_KEY`, `LLM_MODEL` |

Set the required env vars on your [dashboard](https://benchspan.com/dashboard/settings/profile) before running.
