---
title: "benchspan benchmarks"
description: "List available benchmarks and their instances."
---

## List benchmarks

```bash
benchspan benchmarks list
```

Shows all benchmarks available to your account (public + any private benchmarks).

## List instances

```bash
benchspan benchmarks instances <benchmark_name>
```

Shows all instances and named subsets for a benchmark. Useful for understanding what's in a benchmark before running it.

```bash
# See all SWEbench subsets
benchspan benchmarks instances swebench

# See agent-healthcheck subsets
benchspan benchmarks instances agent-healthcheck
```
